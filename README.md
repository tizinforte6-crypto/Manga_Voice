# 📚 Manga Voice Assistant (manga_vose)

Локальное приложение на Python для автоматизации озвучки страниц манги. Программа позволяет выделять область экрана, автоматически распознавать русский текст с картинки, генерировать реалистичную озвучку и управлять проектами через базу данных.

---

## 🛠 Технологический стек

* **Интерфейс:** Tkinter (GUI)
* **База данных:** MongoDB (`pymongo`)
* **Распознавание текста (OCR):** `PaddleOCR` (с поддержкой GPU-ускорения)
* **Синтез речи (TTS):** Локальная нейросеть `Silero TTS` (через PyTorch)
* **Работа с медиа:** `MoviePy`, `Pillow`, `OpenCV`

---

## 🚀 Инструкция по установке и запуску

Рекомендуется запускать на операционной системе Windows с видеокартой NVIDIA (протестировано на RTX 4060) для аппаратного ускорения нейросетей.
